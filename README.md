# Run and deploy your ZEN AI Studio app

This contains everything you need to run your app locally.

View your app in ZEN AI Studio: https://ai.studio/apps/drive/1vf-x2P5KRrDwoZi00ZeOra4FLanoD_zE

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:  
   `npm install`

2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key

3. Run the app:  
   `npm run dev`
